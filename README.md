# Project Management Application

A React 17 application for managing projects with PrimeReact, Tailwind CSS, Storybook, and FluentUI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run Storybook:
```bash
npm run storybook
```

## Technologies

- React 17
- PrimeReact
- Tailwind CSS 3
- Storybook (Node 18 compatible)
- FluentUI 8
- React Router
